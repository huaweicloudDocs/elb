# 删除负载均衡器<a name="DeleteLoadBalancer"></a>

## 功能介绍

删除负载均衡器。

## 接口约束

删除load balancer前必须先删除与其关联的所有的listeners。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=DeleteLoadBalancer)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
