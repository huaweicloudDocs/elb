# 查询转发规则列表<a name="ListL7Rules"></a>

## 功能介绍

查询转发规则列表。

## 接口约束

-   分页查询使用的参数为marker、limit、page\_reverse。

-   marker和page\_reverse只有和limit一起使用时才会生效，单独使用无效。


## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=ListL7Rules)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

GET /v3/\{project\_id\}/elb/l7policies/\{l7policy\_id\}/rules
