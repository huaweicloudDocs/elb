# 查询规格列表<a name="ListFlavors"></a>

## 功能介绍

查询租户在当前region下可用的负载均衡规格列表。

## 接口约束

-   分页查询使用的参数为marker、limit、page\_reverse。

-   marker和page\_reverse只有和limit一起使用时才会生效，单独使用无效。


## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=ListFlavors)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
