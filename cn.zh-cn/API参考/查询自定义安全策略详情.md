# 查询自定义安全策略详情<a name="ShowSecurityPolicy"></a>

## 功能介绍

查询自定义安全策略详情。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=ShowSecurityPolicy)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
