# 变更负载均衡器计费模式<a name="ChangeLoadbalancerChargeMode"></a>

## 功能介绍

负载均衡器计费模式变更，当前只支持按需计费转包周期计费。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=ChangeLoadbalancerChargeMode)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

POST /v3/\{project\_id\}/elb/loadbalancers/change-charge-mode

## 请求示例

将按需计费负载均衡器转包周期计费

```
POST https://{ELB_Endpoint}/v3/060576782980d5762f9ec014dd2f1148/elb/loadbalancers/change-charge-mode

{
  "loadbalancer_ids" : [ "cbf314d0-d52d-4c86-9ad9-95cbf47478cb" ],
  "charge_mode" : "prepaid",
  "prepaid_options" : {
    "period_type" : "year",
    "period_num" : 1,
    "auto_pay" : true,
    "auto_renew" : false
  }
}
```
