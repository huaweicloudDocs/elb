# 查询配额详情<a name="ShowQuota"></a>

## 功能介绍

查询指定项目中负载均衡相关的各类资源的当前配额。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=ShowQuota)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

GET /v3/\{project\_id\}/elb/quotas
