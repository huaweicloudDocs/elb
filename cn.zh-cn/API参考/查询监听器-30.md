# 查询监听器<a name="zh-cn_topic_0141008279"></a>

## 功能介绍<a name="zh-cn_topic_0096561541_zh-cn_topic_0049139640_section63419476"></a>

查询监听器列表。支持过滤查询和分页查询。

可以通过监听器ID、协议类型、监听端口号、关联的后端云服务器的IP等查询监听器。
