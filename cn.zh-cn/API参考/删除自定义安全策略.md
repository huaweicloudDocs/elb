# 删除自定义安全策略<a name="DeleteSecurityPolicy"></a>

## 功能介绍

删除自定义安全策略。

## 接口约束

-   已绑定了监听器的自定义安全策略不能被删除。


## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=DeleteSecurityPolicy)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
