# 更新证书<a name="UpdateCertificate"></a>

## 功能介绍

更新证书。

## 接口约束

如果待更新证书的域名不为空，且被监听器使用；则不允许更新该证书的域名为空，否则系统会返回409响应。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=ELB&api=UpdateCertificate)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。
